Toaster Test Tools for Mono

Toaster is a set of unit testing tools modeled after the Microsoft MSTest tools included in Visual Studio. It is built with Mono 3.x and currently supports OSX and Linux. It can be used to create unit or functional tests, and it's designed to be a command line tool first and foremost.

Toaster includes test ordering attributes to allow you to run your tests in a specific order. This makes it easier to create functional tests that require large amounts of setup and teardown.
